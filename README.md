# Social Feed - Full Stack Social Network

A complete full-stack social network application built with React, Spring Boot, MongoDB, and JWT authentication.

## 🚀 Project Structure

| Layer | Technology |
|-------|-----------|
| Frontend | React 18, Bootstrap 5, CSS3 |
| Backend | Spring Boot 3, Spring Security, Java 17 |
| Database | MongoDB |
| Auth | JWT (JSON Web Tokens) |

## ✨ Features

- JWT Authentication (Login/Register)
- Role-based access control
- Create posts with beautiful UI
- Like/unlike posts with animations
- Add comments
- Dark/Light theme toggle
- Fully responsive design

## 🚀 How to Run

1. Start MongoDB (MongoDB Compass)
2. cd backend && .\mvnw.cmd spring-boot:run
3. cd frontend && npm start
4. Open http://localhost:8081
